# Editing Passes

## Humanity edit
- Remove AI tells
- Remove performative polish
- Keep a human cadence
- Make it sound written, not generated

## Zinsser edit
- Cut clutter
- Prefer verbs
- Shorten the lead
- Keep only the words that pull weight

## Stop rules
- Stop if the next pass only changes tone
- Stop if shortening hurts rhythm
- Stop if the piece is already clear and alive
