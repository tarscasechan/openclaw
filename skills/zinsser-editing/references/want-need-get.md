# Want / Need / Get

Use this as the first planning frame for an essay or post.

## Want
- Start with the bold claim.
- Open curiosity.
- Name what the reader wants or fears.

## Need
- Name the gap.
- Say what must change.
- Explain the pain of the old shape.

## Get
- Show the new shape.
- Say what it is.
- Say what it is not.
- Note new tradeoffs or problems it introduces.

## Good use
- This is especially useful before sentence-level editing.
- If the structure is weak, fix structure before polishing prose.
