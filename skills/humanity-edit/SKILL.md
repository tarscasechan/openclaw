# Humanity Edit

## Goal

Make the draft sound written by a person.

## Rules

- Remove AI tells.
- Remove performative polish.
- Keep cadence and idiom.
- Leave at least one human sentence intact if it carries the piece.
- If a line feels generated, cut or rewrite it.

## Output

A draft that feels lived-in, not synthetic.
