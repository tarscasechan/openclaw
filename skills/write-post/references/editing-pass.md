# Editing Passes

## Humanity edit
- Remove AI tells
- Remove performative polish
- Keep a human cadence
- Keep or restore personal pronouns when the piece is personal or reader-facing
- Make it sound written, not generated

## Zinsser edit
- Cut clutter
- Prefer verbs
- Shorten the lead
- Keep only the words that pull weight

## Read-aloud pass
- Check whether the tightened piece still sounds like someone wrote it.
- Restore `I`, `you`, or `we` if clarity edits made the piece feel unowned.
- Keep concrete texture when it carries the rhythm.
- Stop if further polish would make the piece colder.

## Order
- Fix structure before line edits.
- Run the humanity edit before the Zinsser edit.
- Run the read-aloud pass after Zinsser.
- Keep a strong original line if tightening would flatten it.

## Stop rules
- Stop if the next pass only changes tone
- Stop if shortening hurts rhythm
- Stop if the piece is already clear and alive
