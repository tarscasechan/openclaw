# Zinsser Editing

## Core rule

Edit for clarity first. Keep the voice alive, but cut anything that does not carry weight.

## Workflow

1. Check structure first: can the piece be framed as **want / need / get**?
2. Read once for the point.
3. Trim clutter: filler, throat-clearing, abstractions, hedges, repeated ideas.
4. Prefer verbs over nouns, nouns over decorations.
5. Keep the draft human: preserve rhythm, idiom, and personality.
6. Stop when another pass would only make it shinier, not better.

## What to push toward

- short, direct openings
- active verbs
- concrete nouns
- one idea per sentence when possible
- paragraphs that earn their keep
- progressive disclosure: lead with the point, then deepen only if needed

## What to cut

- self-congratulating phrasing
- inflated transitions
- stacked qualifiers
- passive constructions that hide the actor
- repetition that only restates the previous line
- cleverness that slows the reader down

## Output standard

Return the edited prose with the same meaning, less noise, and more life.
If a change weakens the voice, keep the stronger original.
If the draft is already clean, make only small surgical edits.
If structure is off, fix that before polishing sentences.
